## Quick Start

1. Enable KiCad IPC API: Preferences → Plugins → Enable IPC API
2. Run bridge ping test:
   `& "C:\\Program Files\\KiCad\\10.0\\bin\\python.exe" bridge/kicad_bridge.py --ping`
3. Start Claude Code: `claude` in your project folder
